Enunciado: Crea una página web con un vector de seis componentes (true, 5, false, "hola", "adios", 2). Con el uso de operadores, comprobar si:
  • El cuarto componente es mayor o no que el tercero
  • Realizar un OR entre el primer y el tercer componente
  • Realizar un AND entre el primer y tercer componente.
  • Sumar, restar, multiplicar, dividir y calcular el modulo entre el segundo y el sexto componente.
